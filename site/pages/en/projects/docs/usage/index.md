# clandro-core-package Usage Docs

<!-- @ARK_DOCS__HEADER_PLACEHOLDER@ -->

The [`clandro-core`](https://github.com/cl-andro/clandro-core-package) package provides utils and libraries for Clandro core. It provides core functionality for the Clandro app that must exist in all Clandro execution environments including forks, and is used by other Clandro packages and also 3rd party packages.
